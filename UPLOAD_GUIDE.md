# GitHub 업로드 안내

이 폴더 안의 파일을 새 GitHub 저장소 루트에 그대로 업로드하면 됩니다.

## 업로드할 파일

- `index.html`
- `style.css`
- `app.js`
- `README.md`
- `images/` 폴더 전체

## 추천 저장소 이름

`dashboard-grade3`

## GitHub Pages로 공개하기

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 파일과 `images/` 폴더를 모두 업로드합니다.
3. 저장소의 `Settings`로 이동합니다.
4. `Pages` 메뉴에서 `Deploy from a branch`를 선택합니다.
5. Branch는 `main`, 폴더는 `/root`를 선택하고 저장합니다.

잠시 후 `https://iooi303648-bot.github.io/dashboard-grade3/` 주소로 확인할 수 있습니다.
