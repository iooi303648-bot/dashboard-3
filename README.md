# 아이스크림 AI 디지털 교육자료 대시보드 안내 카드뉴스

아이스크림 AI 디지털 교육자료의 대시보드 확인 방법을 안내하는 정적 웹 카드뉴스입니다.

## 구성

- `index.html`: 10장 카드뉴스 화면
- `style.css`: 기존 카드뉴스 스타일 기반 디자인
- `app.js`: 슬라이드 이동, 모바일 스와이프, 카드 이미지 다운로드
- `images/`: 대시보드 화면 예시 이미지

## 실행

브라우저에서 `index.html`을 열면 바로 확인할 수 있습니다.

카드 PNG 다운로드 기능은 `html2canvas` CDN을 사용하므로 인터넷 연결이 필요합니다.
